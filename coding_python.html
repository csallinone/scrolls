-   Pre-built (Syntax, Keywords, Operators) + 

-   Pre-built and custom Functions, Classes, instances + 

-   user Data.

o   Translator evaluates expression and always return a single value

o   Expression can be:

-   Standalone (one-time execution).  

-   Structured and use flow controls like:   

o   conditionals (if, elif, else),

o   Error handling

-   Data: Referred to user data. (data is NOT keywords or syntax) 

o   Data can be hard coded in expression or stored in external data files.

o   Data can be categorized in data types, using functions like str, int, list, dict etc. 

o   Sometimes Data conversions may perform for compatibilities among other data

o   Variables are used in code to store a single data value.

Verify installation:

Note: Once installed, python expressions can run from both python 'IDLE' or 'OS' shells:

 

# check version:

where python                                       # or '/which python' for macOS/Linux

python --version                                   # or 'python3 --version' for macOS/Linux

 

# Check installation path:

import sys; print(sys.executable)                  # IDLE shell

python -c "import sys; print(sys.executable)"      # OS command prompt

# Check Pre-Built Libraries and Modules

python -c "help('modules')"                        # List all Modules

 

List Functions in a Module (e.g., math):
python -c "help('math')"                           # lists all functions and classes in 'math' module

 

Find Module File Path (e.g., os module):
python -c "import os; print(os.__file__)"

# Python comes with 3rd party packages:
python -m pip list  # list installed packages and versions:

 

python -m pip install <package>

python -m pip uninstall <package>

python -m pip install --upgrade <package>
#Example:
python -m pip install setuptools

python -m pip uninstall setuptools

python -m pip install --upgrade setuptools

python -c "print('hi, world')"
